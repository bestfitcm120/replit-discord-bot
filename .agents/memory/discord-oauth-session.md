## Session approach
HMAC-signed cookies (itsdangerous in Python, crypto.createHmac in Node.js). No DB session table. 7-day expiry. Cookie name: `modbot_session`.

**Why:** Simpler than server-side sessions; works across bot restarts with no cleanup needed.

**How to apply:** Pass SESSION_SECRET env var at runtime. Never hardcode. Set SECURE_COOKIES=true in production (HTTPS).

## Generated React Query hooks
All hooks generated by Orval require `queryKey` to be explicitly passed in the options object.

**Pattern:**
```ts
useGetMe({ query: { queryKey: getGetMeQueryKey(), retry: false } })
useListGuilds({ query: { queryKey: getListGuildsQueryKey(), enabled: !!user } })
```

**Why:** Orval v8 + TanStack Query v5 makes queryKey required in the typed options interface. Forgetting it causes TS2741 errors.

**How to apply:** Always import the matching `getXxxQueryKey()` function alongside its hook. Never skip it.
